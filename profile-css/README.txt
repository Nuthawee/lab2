เขียนสะท้อนผลการเรียนรู้
    ได้รู้จักการใช้ html(Hypertext Markup Language) ในด้านของ semantic ที่ให้ความหมายกับโครงสร้างของ web page เช่น <header>, <main>, <section>, <footer> 
นอกจากนี้ยังได้ใช้ tag ต่างๆช่วยให้มีความเคยชินกับการใช้งานมากขึ้นทั้ง heading, paragraph, table, list, form, link, image เเละจากการใข้ช่วยให้รู้ว่า tag ไหนสามารถใช้เเบบไหนได้บ้าง
เช่น inline element, box element อย่างสุดท้ายได้รู้จักใช้ url เพื่อเชื่อมถึงไฟล์อื่นๆ
